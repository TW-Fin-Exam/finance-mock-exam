# 金融證照模擬測驗系統

仿電腦應試的互動模擬考網站，涵蓋三項測驗：

- **金融市場常識與職業道德**（100 題／60 分／70 分及格）
- **永續發展基礎能力測驗**（80 題／90 分／70 分及格）
- **證券商高級業務員（高業）**：證券交易相關法規與實務、投資學、財務分析

## 功能

- 首頁選測驗 → 模擬考（正式題數＋倒數計時＋交卷結算）或自訂練習（分章、即時解析）
- 隨機抽題，反覆練不重複；成績單＋逐題錯題檢討（各選項解析）
- 題庫共 840 題（含各選項解析與依據）

## 使用

線上版：開啟 GitHub Pages 網址即可（免安裝、免登入）。
單機版：下載 `index.html`，用瀏覽器開啟即可（完全離線、無外部依賴）。

## 聲明

題目為依各測驗官方公告命題範圍自行編寫之練習題（非官方題庫原題）；
考試制度依 2026 年（民國 115 年）官方公告設定，實際規定請以主辦單位（證基會、金研院）最新公告為準。
